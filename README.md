# Dual Principal Component Pursuit
Single subspace learning with some outliers.



## Requirements
- [Yalmip](https://yalmip.github.io/) and [sdpt3](http://www.math.nus.edu.sg/~mattohkc/sdpt3.html) for solving l1 minimization problem;
